# Student Management System (OOP + STL)

Manages student records using a `Student` class and a `vector<Student>`,
demonstrating how OOP and STL containers work together, including sorting
with STL algorithms.

## Concepts Used
- Classes
- `vector<T>`
- STL `<algorithm>` (`sort`)
- Lambda expressions

## How to Run
```bash
g++ -std=c++17 student_management.cpp -o student_management
./student_management
```
