# 03. Operators

## Concepts Covered
- Arithmetic, relational, logical, bitwise, assignment operators
- Ternary operator
- Increment/decrement operators
- `sizeof` operator

## Files

| File | Description |
|------|--------------|
| `01_arithmetic_operators.cpp` | +, -, *, /, % |
| `02_relational_logical_operators.cpp` | Comparisons and boolean logic |
| `03_bitwise_operators.cpp` | Bit-level manipulation |
| `04_assignment_operators.cpp` | Shorthand assignment operators |
| `05_ternary_and_increment.cpp` | Ternary operator, pre/post increment-decrement |

## Key Takeaways
- Operator behavior mirrors C, with the addition of operator overloading for user-defined types (see [17_Operator_Overloading](../17_Operator_Overloading)).
- `&&` and `||` short-circuit, just like in C.
