# 05. Conditional Statements

## Concepts Covered
- `if`, `if-else`, `else if` ladder
- Nested conditionals
- `switch-case`

## Files

| File | Description |
|------|--------------|
| `01_if_else.cpp` | Basic if / if-else |
| `02_else_if_ladder.cpp` | Grading system with multiple conditions |
| `03_nested_if.cpp` | Conditions depending on other conditions |
| `04_switch_case.cpp` | Multi-way branching |

## Key Takeaways
- Same semantics as C, but conditions naturally work with `bool`.
- `switch` in C++ also supports scoped variables inside cases (with braces).
