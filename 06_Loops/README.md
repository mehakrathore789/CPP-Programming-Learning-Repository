# 06. Loops

## Concepts Covered
- `for` loop
- `while` loop
- `do-while` loop
- Range-based `for` loop (C++11) — the modern way to iterate over containers
- `break` and `continue`

## Files

| File | Description |
|------|--------------|
| `01_for_loop.cpp` | Printing numbers 1 to 10 |
| `02_while_loop.cpp` | Summing digits of a number |
| `03_do_while_loop.cpp` | Menu that always runs at least once |
| `04_range_based_for.cpp` | Iterating over a vector the modern way |
| `05_break_continue.cpp` | Loop control statements |

## Key Takeaways
- Range-based `for` (`for (int x : container)`) is preferred in modern C++ for readability.
- Use `for (auto& x : container)` to modify elements in place without copying them.
