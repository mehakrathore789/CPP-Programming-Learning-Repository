# 24. Practice Questions

Solved practice problems combining core C++ and STL concepts, similar to
what you'd encounter in interviews or coursework.

## Files

| File | Description |
|------|--------------|
| `01_swap_two_numbers.cpp` | Swap using references vs a temp variable |
| `02_prime_number_check.cpp` | Check whether a number is prime |
| `03_bubble_sort.cpp` | Sort an array using Bubble Sort |
| `04_binary_search.cpp` | Search a sorted vector using Binary Search |
| `05_word_frequency_counter.cpp` | Count word frequencies in a sentence using `map` |
| `06_vector_of_objects.cpp` | Storing and querying custom objects in a vector |
| `07_check_leap_year.cpp` | Determine whether a year is a leap year |
| `08_operator_overload_fraction.cpp` | A `Fraction` class with overloaded arithmetic operators |

## Key Takeaways
- Try solving each problem yourself before checking the solution.
- Notice how STL containers (`map`, `vector`) simplify problems that would
  need manual array management in C.
